A multiple counter management interface where users can:

1. Create multiple named counters
2. Increment individual counters by clicking on the a counter's button.increment button
3. Remove specific counters, by clicking the button#remove button
4. Randomly increment one of the existing counters with the button#randomIncrement button
5. Add new counters to the list with the button#add button
6. See a running total of all counter values, displayed in a span#total
7. Edit the overall title of the counter collection with the common-input#title input
