A small guestbook for visitors to sign their names, and leave a guestbook entry.

Users submit

1. A name
2. A text entry

The guestbook displays all entries, sorted by entry timestamp, in descending order (newest on top).

Flow:

1. User enters a name
2. User enters an optional text entry
3. User clicks "Sign Guestbook"
4. User's entry is displayed in the guestbook, with the newest entries on top
