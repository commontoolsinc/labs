# CFC Trusted Component Examples

This gallery contains 52 untrusted host examples that embed canonical trusted UI
surfaces:

- 30 send/publish examples in `send-publish-examples.tsx`
- 14 disclaimer, influence, provenance, and fact-check examples in
  `disclaimer-examples.tsx`
- 4 process-oriented examples in `process-examples.tsx`
- 4 recipient-confirmation and redacted-release examples in
  `confirmation-release-examples.tsx`

The aggregate interactive entry point is `main.tsx`.
